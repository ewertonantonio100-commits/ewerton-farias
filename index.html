Sob o capô

As decisões de engenharia por trás de cada tela.

engineering.log — produção
[ SEGURANÇA ]OAuth 2.0 + Edge Functions com service_role: o token da OLX nunca chega ao frontend. Leitura só server-side, RLS sem policy de exposição.
[ MIGRAÇÃO ]Camada de normalização para migrar dados legados do Firebase v1 → v2 sem quebrar histórico de clientes reais.
[ ARQUITETURA ]Multi-tenant com Row Level Security no Supabase — cada loja enxerga só os próprios dados (loja_id = auth.uid()).
[ CI/CD ]GitHub → Vercel: deploy a cada push, build script custom, domínios próprios com HTTPS. Pipeline inteiro rodando de um terminal Termux.
[ PAGAMENTOS ]Stripe em live mode (assinatura mensal/anual) + trial in-app + e-mail transacional via Resend. Também já operei Mercado Pago na v1.
