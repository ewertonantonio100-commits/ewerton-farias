Checklist técnico

decisões verificadas em produção

Credencial sensível nunca chega ao navegador

O token da OLX é lido e renovado apenas por Edge Function com service role. A tabela de conexões não tem policy de leitura para o front-end.

Cada loja enxerga só os próprios dados

Multi-tenant garantido por Row Level Security no PostgreSQL, na camada do banco — não por filtro na aplicação, que é onde esse tipo de regra costuma vazar.

Migração sem perder cliente no caminho

A virada de arquitetura do Authon usou uma camada de normalização entre as duas versões, para que dados antigos continuassem legíveis enquanto o sistema novo assumia.

Peso de página tratado como requisito

Os módulos de PDF passaram a carregar sob demanda. O pacote inicial caiu de cerca de 1,3 MB para 792 KB — diferença que se sente em oficina com internet ruim.

Publicar não depende de mim estar na frente do computador

Push no GitHub dispara o build e o deploy na Vercel, com domínio próprio e HTTPS. O mesmo fluxo roda dos dois projetos.
